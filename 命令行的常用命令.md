# 命令行的常用明亮；

- 按住shift在文件中邮件，可以出来命令行选项
- cd 进入文件夹；cd..可以退出当前文件夹；
- ctrl+C 连续两次，可以退出交互环境；
- mkkdir broszhu 创建一个叫broszhu的文件夹.
- cd broszhu 后可以输入git init;可以创建一个仓库；
- ls -al 可以看所有文件夹；(ls也可以查看， ls-al是查看所有的)

- npm init 初始化一个仓库
